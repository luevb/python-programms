---


## Описание алгоритма

### Что такое шейкерная сортировка?
Шейкерная сортировка (также известная как коктейльная сортировка или двунаправленная пузырьковая сортировка) - это улучшенная версия пузырьковой сортировки.

### Принцип работы:
1. **Проход слева направо** - "всплывают" самые большие элементы  
   (как в пузырьковой сортировке)
2. **Проход справа налево** - "тонут" самые маленькие элементы  
   (это отличие от обычной пузырьковой)
3. Границы сортировки сужаются с обоих концов  
   (левая граница увеличивается, правая уменьшается)
4. Процесс повторяется, пока массив не будет отсортирован  
   (до встречи границ)

├── .venv/ # Виртуальное окружение Python

### Назначение файлов:

| Файл | Что делает |

| `src/shaker_sort.py` | Содержит сам алгоритм сортировки |
| `src/utils.py` | Функции для ввода с клавиатуры, |

## Установка и запуск

### Требования
- Python 3.6 или выше
- Дополнительные библиотеки не требуются

### Клонирование репозитория
```bash
git clone https://github.com/ваш-логин/lab_2_sort.git
cd lab_2_sort
```

### Запуск программы  
```bash
# Из корневой папки проекта
python src/main.py
```
### Запуск тестов
```bash
# Отдельно от основной программы
python tests/test_shaker_sort.py
```
